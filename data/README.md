# PQC Regulatory Deadlines & Algorithm Data

This directory contains structured CSV data extracted from global cryptographic compliance frameworks, including NIST IR 8547 and the EU Coordinated PQC Roadmap[cite: 3, 5]. 

This data is intended for automated ingestion into compliance tracking tools, GRC (Governance, Risk, and Compliance) platforms, or internal project management dashboards to track obsolescence risks.

## Files included

* `pqc-regulatory-deadlines.csv` - Global compliance deadlines for RSA/ECC deprecation, CBOM generation, and final disallowance[cite: 3, 5, 10].

**Data source:** [The Post-Quantum Migration Playbook](https://agileleadershipdayindia.org/blogs/post-quantum-cryptography-crypto-agility-ciso/post-quantum-cryptography-crypto-agility-ciso.html)[cite: 5]. 

**Maintenance note:** Updates to this dataset should be submitted via PR when regulatory bodies (e.g., NIST, ENISA, NSA) publish revised timelines or parameter updates.
